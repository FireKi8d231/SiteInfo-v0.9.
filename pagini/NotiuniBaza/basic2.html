Structuri de decizie

  • Structurile de decizie sau alternative execută o instrucțiune o singură dată dacă și numai dacă condiția respectivă este adevărată.


  • Exista 2 instructiuni de decizie: if/else si switch.

I. Instructiunea if/else

  • Această este cea mai utilizată instructiune de decizie și se executa astfel:Se verifică dacă o condiție este adevarata. În cazul in care este adevarată, se va executa o instructiune, in cazul contrar se va executa alta instructiune, sau in anumite cazuri nu se va executa nimic.

Sintaxa:


Varianta I Varianta II
if (conditie)
{
(instructiune1)
}
else{
(intructiune2)
}		 if (conditie)
{
(instructiune)
}

  • Putem observa ca instructiunea else nu este obligatorie.In acest caz ,daca conditia nu este adevarata nu se va executa nici o instructiune si doar va iesi din structura.

Cerinta:

Fiind dată nota unui elev să se afișeze dacă acesta este corigent sau promovat.

Exemplu:

Intrare: 5

Ieșire: promovat

 6 |if (n < 5) cout << "corigent";//daca nota elevului este mai mica decat 5 programul va scrie corigent
 7 |else cout << "promovat";//daca nu programul va scrie promovat

II. Instructiunea switch

  • Instrucțiunea switch permite executarea unor instrucțiuni, în funcție de egalitatea unei expresii cu anumite valori numerice constante:

Sintaxa:

switch(expresie)
{
case NrConstant1:
Instructiune1;
break;
case NrConstanta2:
Instructiune2;
break;
.....
case NrConstantN:
InstructiuneN;
break;
default:
Instructiunedefault;
break;
}

  • Instructiunea default nu este obligatorie, aceasta fiind executata numai daca expresia nu este egala cu nici o valoare constanta.
    Instructiunea brake de asemenea nu este obligatorie. Daca aceasta nu este folosita intr-un case, programul va executa comenzile din toate cazurile viitoare pana la urmatorea instructiune brake intalnita sau pana la terminarea instructiunii switch.
