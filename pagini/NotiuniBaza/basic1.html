Structuri repetitive


  • Structurile repetitive execută o instrucțiune de un anumit numar de ori sau cat timp o condiție este adevărată.

  • Structurile repetitive sunt de doua tipuri:

    -cu numar cunoscut de pasi (stim de la inceput de cate ori se va executa insrtuctiunea)
     -cu numar necunoscuti de pasi (instructiunea se va executa cat timp o conditie este adevarat)

  • Structurile cu numar necunoscuti de pasi sunt de asemenea de doua tipuri:

    -cu test initial (verificam conditia si pe urma executam instructiunea)
    -cu test final (executam instructiunea si pe urma verificam conditia)

  • Exista 3 instructiuni repetitive: while, do/while si for.

I. Instructiunea while

  • Aceasta instructiune are numar necunoscuti de pasi si test initial.

Sintaxa:

while (conditie)
{
(instructiune)
}

II. Instructiunea do/while

  • Aceasta instructiune are numar necunoscut de pasi si un test final.

Sintaxa:

do{
(instructiuni)
}
while(conditie)

III. Instructiunea for

  • Aceasta instructiune are numar necunoscuti de pasi si test final.

Sintaxa:

for((declaram variabila);(conditie);(instructiune1))

{
(instructiune2)
}

  • Declararea variabilei se va executa o singura data la intrarea in for.

Cerinta:

Se citesc numere de la tastatură până la apariția lui zero. Să se determine câte dintre ele erau pare.

Restrictii si precizari:

    numerele citite aparțin intervalului (-1.000.000,1.000.000); în cazul în care niciun număr nu este par se va afișa mesajul NU EXISTA.

Intrare:1 2 3 4 5 6 0

Ieșire: 3

 8 |cin >> x;//introducem un numar nou x
 9 |}
 10|if (nr == 0) cout << "NU EXISTA";//daca nr are valoarea 0 inseamna ca nu a fost gasit nici un nr par, deci vom afisa "NU EXISTA"
