1.Keybind-uri Generale

[CTRL+C] - copiaza ce ai selectat
[CTRL+V] - da paste la ce ai selectat
[CTRL+A] - selecteaza tot ce se poate de pe aplicatie/fereastra pe care esti
[ALT+TAB] - arata aplicatiile deschise si poti sa schimbi intre ele
[CTRL+X] - sterge si copiaza ce ai selectat
[ALT+F4] - inchide aplicatia activa

2.Keybind-uri in Visual Studio

[CTRL+SHIFT+N] - creaza un nou fisier
[CTRL+Z] - undo
[CTRL+SHIFT+Z] - redo
[F5] - run

3.Keybind-uri in Code::Blocks

[CTRL+N] - creaza un nou fisier
[F9] - build si run
[CTRL+Z] - undo
[CTRL+SHIFT+Z] - redo

4.Keybind-uri in OnlineGDB

[CTRL+M] - creaza un nou fisier
[F9] - run
[CTRL+Z] - undo
[CTRL+SHIFT+Z] - redo
