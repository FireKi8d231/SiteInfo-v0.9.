Aflarea minimului și a maximului

  • Pentru a afla maximul și minimul a doua numere, vom folosi structura if.
  • Afisarea minimului dintre doua numere a si b:

Metoda Ⅰ

 1 |#include <iostream>//introducem bibloteca
 2 |using namespace std;
 3 |int main() {
 4 |int a, b;//declaram numerele
 5 |cin >> a >> b;//inroducem numerele
 6 |if (a < b)cout << a;//inseamna ca minimul este a deci il afisam
 7 |else cout << b;//daca nu a este minimul inseamna ca este b-ul deci il afisam
 8 |return 0;
 9 |}

Metoda II

 1 |#include<iostream>
 2 |using namespace std;
 3 |int main() {
 4 |int a, b;
 5 |cin >> a >> b;
 6 |cout << min(a, b);
 7 |return 0;
 8 |}

  • Desigur ca acest cod poate fi folosit si pentru aflarea maximului, schimband "<" cu ">" in metoda Ⅰ si schimband "min" cu "max"din metoda II

#354 n_maxim(Problema pbinfo)

Cerința

Se dau n numere întregi. Calculaţi cel mai mare dintre cele n numere date.

Restricții și precizări:

1 ≤ n ≤ 1000

cele n numere citite vor avea cel mult 9 cifre

Exemplu

Intrare:5 7 6 9 6 8

Ieșire: 9 


 1 |#include<iostream>// introducem biblioteca
 2 |using namespace std;
 3 |int main() {//programul principal
 4 |int n, maxi, x,a;//declaram variabile
 5 |cin >> a;//citim variabila n
 5 |cin >> x;//citim prima valoare data
 6 |maxi = x;// ii atribuim lui maxi prima valoare citita
 7 |for (int i = 2; i <= n; i++) {
 8 |cin >> x;
 9 |if (maxi < x) maxi = x;//daca una dintre numerele citite este mai mare decat maxi, variabila maxi va primi
 10|valoarea acelui numar
 11|}
 12|cout << maxi;//afisam maximul
 13|return 0;
 14|}
