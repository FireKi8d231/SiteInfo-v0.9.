Divizibilitate/Numere prime

Pentru a afla divizorii unui numar sau daca acesta este prim, vom folosi algoritmii de mai jos.

Algoritm 
 1 | #include <iostream> // includ biblioteca
 2 | using namespace std;
 3 |int main ()
 4 |{
 5 |int n, d, ok=1;//consideram ca la inceput numarul este prim
 6 |cin>> n;
 7 |if(n< 2) ok=0;//2 este primul numar prim si singurul nr prim par
 8 |for(d=2; d*d<=n;d++)
 9 |if(n%d==0) ok=0;//impartim numarul la toti divizorii posibili de la 2 pana la jumatatea numarului n; 
   |daca n se imparte exact la unul dintre divizori, atunci numarul nu este prim (ok=0)
 10|if(ok==1) cout<<"prim";//daca ok a ramas cu valoarea 1, atunci n este prim si se va afisa un mesaj corespunzator
 11|if(ok==0) cout<<"nu este prim";/ daca ok a ramas cu valoarea 0,
   | atunci n nu este prim si se va afisa un mesaj corespunzator
 12|return 0;
 13|}

#387 Suma Divizorilor Impari(Problema pbinfo)

Cerința

Se citește un număr natural n. Să se determine suma divizorilor impari ai săi.

Restricții și precizări:

0 < n < 1.000.000.000

Exemplu

Intrare:30

Ieșire: 24

Divizorii impari lui 30 sunt 1, 3, 5 și 15. Suma lor este 24.

 1 |#include<iostream>// introducem biblioteca
 2 |using namespace std;
 3 |int main(){//programul pricipal
 4 |int n;int d;int s=0 ;//declaram variabile si dam sumei valoarea initiala 0
 5 |cin>>n;//citim variabila n
 6 |for(d = 1;d * d < n;d++){
 7 |if(n%d==0)//daca n ete divizibil cu d, vom verifica paritatea variabilei d
 8 |{
 9 |if(d%2==1) s=s+d;//daca d nu este divizibil cu 2, acesta va fi adunat la b=suma
 10|if((n/d)%2==1 and d!=n/d) s=s+n/d;
 11|}
 12|}
 13|cout<<s;//afisam suma
 14|return 0;
 15|}

Ciurul lui Eratostene

  • Ciurul lui Eratostene reprezinta o metoda de determinarea tuturor numerelor prime mai mici sau egale cu o valoare data, n.

Acest algoritm poate fi folosit pentru aflarea tuturor numerelor prime mai mici sau egale cu n. 

 1 |#include<iostream>
 2 |using namespace std;
 3 |int main() {
 4 |  int n;
 5 |  int prim[200002];
 6 |  cin >> n;
 7 |  prim[0] = prim[1] = 1;// dam valoare vectorului de pe pozitia 1 si 0, 1 deoarece 1 si 0 nu sunt prime
 8 |  for (int i = 2; i <= n; i++) {//parcurgem numerele mai mici de n
 9 |      if (prim[i] == 0) {//i nu a fost marcat,deci e prim
 10|          for (int j = i + i; j <= n; j += i)//marcam multiplii lui i
 11|              prim[j] = 1;
 12|      }
 13|  }
 14|  return 0;
 15|}
